# SkillSphere

> A peer-to-peer skill learning platform where users teach what they know
> and learn what they need — with real-time sessions, quizzes, badges, and peer validation.

## Tech Stack
- Frontend: React, Vite, Tailwind CSS, Socket.io-client
- Backend: Node.js, Express.js, MongoDB, Socket.io
- Auth: JWT + bcryptjs
- Deploy: Vercel + Render + MongoDB Atlas

## Run Locally

### Backend
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev

## Features
- Dual role system: teach and learn
- Real-time chat via Socket.io
- Video sessions via Jitsi Meet
- Post-session quizzes with badges
- Peer skill endorsements
- Points and leaderboard
